##### 舰队

### 调用地址

https://api.live.bilibili.com/guard/topList?

需要 Cookies 并验证登录状态(Cookies)；要求应用申请弹幕权限

#### 请求参数
| 字段       | 必选    |传递方式| 类型     | 说明                                       |
| -------- | ----- | ------|-------- | ---------------------------------------- |
| roomid      | true  |GET| int    | 直播房间id                                      |
| ruid        | true  | GET|int    | up主uid                                     |
| page        | true  | GET|int    | 当前页数                                 |
